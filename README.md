# RWQForecast

**RWQForecast (Remote Water Quality Forecast)** is a web application designed to analyse **temporal changes** in water quality within reservoirs and to assess the **spatial distribution** of water quality within a reservoir for a given date. The analysis is based on a combination of **Sentinel 2** satellite data and AI **Machine Learning and Deep Learning** methods.

RWQForecast offers the following options for processing and evaluating satellite data:

* Estimation of **water quality** in reservoirs
* Evaluation of **temporal changes**
* Assessment of the **spatial distribution** of water quality in reservoirs

RWQForecast allows evaluation of the following parameters:

* **ChlA** – Chlorophyll-a concentration (µg.l⁻¹)
* **TSS** – Total suspended solids (mg.l⁻¹)
* **PC** – Phycocyanin concentration (µg.l⁻¹)

The system's functionality is designed to minimise user input requirements. The web application handles the user interface, where users simply select a reservoir from a map or list and specify the desired water quality parameters.
